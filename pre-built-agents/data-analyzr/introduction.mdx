DataAnalyzr is an innovative conversational analytics framework that provides access to LLM-integrated advanced data analysis through a conversational interface to derive actionable insights and intuitive visualizations.

With DataAnalyzr you can streamline the complexity of data analytics into a powerful, intuitive, and conversational interface that lets you command data with ease.
Whether you're an experienced data scientist or a business analyst, DataAnalyzr opens up a world of possibilities by transforming raw data into actionable insights and engaging visualizations.

DataAnalyzr supports both [Pythonic](/pre-built-agents/data-analyzr/pythonic-analysis-agent) and [SQL-based analysis](/pre-built-agents/data-analyzr/text-sql-agent), allowing you to choose the best approach for your data analysis needs.

  <Accordion title="Pythonic Data Analysis">

## Editions

DataAnalyzr comes in two distinct editions: open-source and enterprise.

The open-source version is free to use and can be installed using Python’s pip.
It is designed for individual users and small teams that require basic data analysis capabilities.

The enterprise version of DataAnalyzr is a paid subscription service that offers a full suite of features and support from our technical team.
It is designed for larger organizations that require more advanced capabilities and dedicated support.
For more details on enterprise DataAnalyzr visit the [enterprise documentation](/enterprise/data-analyzr/introduction).

| Open Source Edition (OSE)                | Enterprise Edition (EE)                                                                                                          |
|------------------------------------------|----------------------------------------------------------------------------------------------------------------------------------|
| Free to Use                              | Paid Subscription Required                                                                                                       |
| Handles limited data sources             | Handles a wide range of data sources                                                                                             |
| No data preprocessing included           | Preprocesses and cleans data when needed                                                                                         |
| Pythonic and SQL-based analyses          | Pythonic, SQL and NoSQL analyses                                                                                                 |
| Limited to basic features                | Full suite of features, e.g. support for latest LLMs, get generated code and analysis output, save images to any S3 bucket       |
| Support options limited to documentation | Documentation along with support from Lyzr’s tech team to implement feature requests and new data sources                        |
