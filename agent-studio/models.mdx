Lyzr Agent Studio allows you to choose from a large set of Models. Agent Studio has integrated with LLM model providers such as Open AI, Google, Anthropic, Bedrock and DeepSeek. Here are the list of models available:
1. OpenAI
    - Gpt-4o-mini
    - Gpt-4o
    - o1
    - o1-preview
    - o1-mini
    - o3-mini
2. Anthropic
    - Claude-3-5-sonnet-latest
    - Claude-3-5-haiku-latest
3. Google
    - Gemini-2.0-flash-exp
    - Gemini-1.5-flash
    - Gemini-1.5-pro
4. AWS Bedrock
    - Claude-3-5-sonnet
    - Claude-3-5-haiku
    - Claude-3-sonnet
    - Claude-3-haiku
    - Claude-3-opus
    - Llama 3.3 70b instruct
    - Llama 3.2 1b instruct
    - Llama 3.2 3b instruct
    - Llama 3.2 11b vision instruct
    - Llama 3.2 90b vision instruct
    - Mistral-7b instruct
    - Mistral 8x7b instruct
    - Mistral large
    - Mistral small

Choose the relevant model as per your agent & your need.
