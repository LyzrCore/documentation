> Navigate to Agent Builder from the sidebar menu and select Create New Agent.
> ![](/assets/images/lyzr-studio/Createnewagent.png)

Enter a unique name and provide a detailed description of the agent’s purpose.

![](/assets/images/lyzr-studio/NameDescription.png)	Choose an LLM provider from the list and select a suitable model. You can click on "configure icon" to change temperature and Top P.

![](/assets/images/lyzr-studio/Provider_model.png)Define the Agent Role (e.g., Customer Support, Data Fetcher) and provide agent instructions to guide its behavior (e.g., Instructions: "Craft professional, personalized emails based on user input.").&#x20;

Use the Improve button to enhance the agent’s role and instructions, ensuring optimal performance with LLMs (Optional but recommended)

![](/assets/images/lyzr-studio/Improve.png)Add Tools (APIs, databases) and Knowledge Base if needed. (Optional)

Quick guide to add Tools

![](/assets/images/lyzr-studio/Tools.png)Quick guide to add Knowledge base

Provide Examples to guide your agent through a template (Optional)

![](/assets/images/lyzr-studio/Examples.png)Select from additional features like short-term memory, bias checks, and humanizer functions to refine the agent’s capabilities (Optional). Here's a quick guide on how to choose the right features.

![](/assets/images/lyzr-studio/Features.png)Click Save & Test to check responses before publishing.

![](/assets/images/lyzr-studio/Inference_create.png)![](/assets/images/lyzr-studio/Testing.png)

Publish your agent as part of Agent Studio, or copy the JSON & API key and use it as part of your product. (Optional)

![](/assets/images/lyzr-studio/Launchagent.png)
