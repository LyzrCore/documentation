Endpoint: `GET /v1/sessions/{session_id}/summary`

Retrieve a summary of the chat payload for a specific session. This endpoint provides a detailed summary of all interactions between the user and the assistant during the session, formatted as a single payload.

#### Path Parameters

<ParamField path="session_id " type="string" required>
    The unique ID of the session.
</ParamField>

<CodeGroup>
<RequestExample>
```bash Curl
curl -X GET "https://api.example.com/v1/sessions/{session_id}/summary" \
-H "x-api-key: your_api_key_here"
```
```python Python
import requests

session_id = "IITZ"
url = f"https://api.example.com/v1/sessions/{session_id}/summary"
headers = {
    "x-api-key": "your_api_key_here"
}

response = requests.get(url, headers=headers)
```
