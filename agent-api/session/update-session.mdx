Endpoint: `PUT /v1/sessions/{session_id}`

This endpoint updates the details of a specific session based on the provided `session_id`.

#### Path Parameters

<ParamField path="session_id " type="string" required>
    The unique ID of the session to update.
</ParamField>

#### Parameters
<ParamField path="user_id " type="string" required>
 The ID of the user associated with the session.
</ParamField>
<ParamField path="metadata " type="object" required>
 Optional metadata associated with the session.
</ParamField>


<CodeGroup>
<RequestExample>
```bash Curl
curl -X PUT "https://agent.api.lyzr.app/v1/sessions/{session_id}" \
-H "Content-Type: application/json" \
-H "x-api-key: your_api_key_here" \
-d '{
  "user_id": "1234",
  "metadata": {}
}'

```
```python Python
import requests
import json

# Define the API URL and headers
session_id = "IITZ"
url = f"https://agent.api.lyzr.app/v1/sessions/{session_id}"
headers = {
    "Content-Type": "application/json",
    "x-api-key": "your_api_key_here"
}
data = {
    "user_id": "7422",
    "metadata": {}
}

response = requests.put(url, headers=headers, data=json.dumps(data))
```
</RequestExample>
</CodeGroup>
